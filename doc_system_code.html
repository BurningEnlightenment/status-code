Type alias system_error2::system_code

using system_code = status_code<erased<long>>;

An erased-mutable status code suitably large for all the system codes which can be returned on this system.

For Windows, these might be:

- `com_code` (`HRESULT`)  [you need to include "com_code.hpp" explicitly for this]
- `nt_code` (`LONG`)
- `win32_code` (`DWORD`)

So the erased type is LONG on Windows, as that can represent all of the above.

For POSIX, posix_code (int) is possible, so this erased type is int i.e. this is the type alias on POSIX, not the above:

using system_code = status_code<erased<int>>
